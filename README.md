# Logging Package

The Logging package provide a simple and efficient logging for your OmegaCMS applications. Easily record and monitor events, errors, and informational messages. Customize logging levels and outputs to suit your needs. Get started quickly with the logging package and keep your application insights organized.

## Requirements

* PHP 8.2 or later

## Installation via Composer

Add `"omegacms/logging": "^1.0.0"` to the require block in your `composer.json` file and then run `composer install`.

```json
{
    "require": {
        "omegacms/logging": "^1.0.0"
    }
}
```

Alternatively, you can simply run the folowing from the command line:

```sh
composer require omegacms/logging "^1.0.0"
```

If you want to include the test sources, use:

```sh
composer require --prefer-source omegacms/logging "^1.0.0"
```
